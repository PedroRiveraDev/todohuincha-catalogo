### Características

- El cabezal, el marco y la base están fabricados en hierro fundido para mayor durabilidad y reducir vibraciones durante el trabajo.
- Un solo perno ajusta el transportador en paralelo con el cabezal de lijado, facilitando la alineación para un lijado rápido y sencillo.
- La palanca rápida (Fast Lever) permite un ajuste indexado para materiales anchos y estrechos.
- Cuando la pieza es más ancha que el tambor de lijado, la palanca ajusta automáticamente el transportador para un lijado sin fallos.
- La mesa del transportador es más ancha que el tambor para brindar un soporte adecuado a piezas grandes.
- El control inteligente de lijado (Intel- Sanding) monitorea constantemente la carga del motor del tambor y regula automáticamente la velocidad del transportador para mantener la máxima eficiencia sin sobrecarga.
- El tambor de aluminio está mecanizado con precisión y equilibrado dinámicamente para disipar el calor y proteger la superficie de trabajo contra daños.
